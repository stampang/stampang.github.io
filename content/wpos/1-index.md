# 윈도우(Windows) 포스 오류 해결방법

## 서버 연결이 되지 않습니다.
- 현재 실행 중인 모든 웹 브라우저를 종료합니다.
- 시작 버튼을 누릅니다.
- Win+R 키를 눌러 cmd 를 입력하세요.
- 명령 프롬프트가 나타나면 ipconfig /flushdns 를 입력합니다.
- 컴퓨터를 재부팅시켜줍니다.

## 실시간페이지에서 주문내역이 보이지 않습니다.

[다운로드](root_update.zip) 하여 압축을 풀고 Cert_Updater_v1.6.exe를 실행합니다. (재부팅 필수)
